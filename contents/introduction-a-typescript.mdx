# TypeScript : Un Typage Puissant pour JavaScript

TypeScript ajoute des types statiques à JavaScript, permettant de créer des applications plus robustes et maintenables.

## Types de Base

Voici quelques exemples de types en TypeScript :

```ts
let nombre: number = 42;
let texte: string = 'Bonjour, TypeScript!';
let estVrai: boolean = true;
```

## Interfaces

Les interfaces permettent de définir la structure des objets :

```typescript
interface Utilisateur {
  id: number;
  nom: string;
  email: string;
}

const user: Utilisateur = {
  id: 1,
  nom: 'Alpha Diop',
  email: 'alpha@example.com',
};
```

## Pourquoi Utiliser TypeScript ?

- **Typage Statique** : Détectez les erreurs avant l'exécution
- **Meilleure Maintenance** : Code plus lisible et plus structuré
- **Compatibilité** : Compile vers JavaScript standard

_TypeScript rend votre code JavaScript plus puissant et fiable._
