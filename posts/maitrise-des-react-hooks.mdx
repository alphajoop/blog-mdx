# React Hooks : Révolutionnez Votre Développement React

Les React Hooks ont transformé la façon dont nous écrivons des composants React, rendant le code plus simple et plus lisible.

## useState : Gestion de l'État Local

Exemple simple de gestion d'état avec `useState` :

```typescript
import React, { useState } from 'react';

function Compteur() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Vous avez cliqué {count} fois</p>
      <button onClick={() => setCount(count + 1)}>
        Cliquez ici
      </button>
    </div>
  );
}
```

## useEffect : Gestion des Effets de Bord

Utilisation de `useEffect` pour les opérations après le rendu :

```typescript
import React, { useState, useEffect } from 'react';

function ExempleFetch() {
  const [donnees, setDonnees] = useState([]);

  useEffect(() => {
    fetch('/api/donnees')
      .then(reponse => reponse.json())
      .then(resultat => setDonnees(resultat));
  }, []);

  return (
    <ul>
      {donnees.map(item => (
        <li key={item.id}>{item.nom}</li>
      ))}
    </ul>
  );
}
```

## Avantages des Hooks

- **Simplicité** : Code plus concis
- **Réutilisabilité** : Logique facilement partageable
- **Pas de Classes** : Moins de complexité

_Les Hooks ont simplifié le développement React._
