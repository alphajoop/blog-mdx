# Next.js 14 : Le Nouveau Système de Routing

Next.js 14 introduit un système de routing plus puissant et intuitif.

## Structure de Base

Exemple de structure de routing :

```
app/
├── layout.tsx
├── page.tsx
├── articles/
│   ├── page.tsx
│   └── [slug]/
│       └── page.tsx
```

## Routes Dynamiques

Création de routes dynamiques :

```typescript
// app/articles/[slug]/page.tsx
export default function ArticlePage({
  params
}: {
  params: { slug: string }
}) {
  return <h1>Article : {params.slug}</h1>
}
```

## Middleware de Routing

Exemple simple de middleware :

```typescript
import { NextResponse } from 'next/server';
import type { NextRequest } from 'next/server';

export function middleware(request: NextRequest) {
  const path = request.nextUrl.pathname;

  if (path.startsWith('/admin')) {
    return NextResponse.redirect(new URL('/login', request.url));
  }
}
```

## Principaux Avantages

- **Performance** : Routing plus rapide
- **Flexibilité** : Routes dynamiques simplifiées
- **Optimisation** : Chargement côté serveur amélioré

_Next.js 14 transforme l'expérience de routing._
